# ReactNativeFizzbuzz

A test-driven Fizzbuzz application in React Native.

Test frameworks are Jest and Detox.

# Installation

`npm install`

# Running Program

`npm start`

To run on IOS: `react-native run-ios`

# Running Tests

To run the E2E tests, `detox test -c ios`

To run component tests, `yarn test` or `jest src`
